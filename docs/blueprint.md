# **App Name**: Drywall Pro

## Core Features:

- Module Display: Showcase available modules with titles, descriptions, lesson count, and duration.
- Module Access: Enable users to access individual modules and lessons within the Drywall course.
- Resource Downloads: Provide access to downloadable resources such as PDF guides and supplier lists.
- Community Link: Integrate a direct link to the WhatsApp community group for course members.
- Support Link: Offer a direct link to WhatsApp support for course-related inquiries.
- Personalized Learning Path: AI tool to provide learning path recommendations based on skill level.

## Style Guidelines:

- Primary color: Gold (#F5C400), as in the original design, conveys expertise, value, and professionalism.
- Background color: Dark navy blue (#0A1930), creating a sophisticated and focused learning environment. A desaturated version of the gold primary, it provides high contrast against other elements.
- Accent color: Deep purple (#311F5B) complements the gold and navy blue, highlighting calls to action and important information without distracting from the main content.
- Headline font: 'Poppins', a geometric sans-serif for a modern, clean look in headings.
- Body font: 'Inter', to ensure excellent readability for the lesson descriptions.
- Use clear, simple icons in gold to represent different module types and resources, enhancing visual navigation.
- Maintain a card-based layout for modules, ensuring easy navigation and visual consistency, as in the original design.
- Subtle animations and transitions on card hover and button clicks to enhance user engagement.
